# 测试文档

---

## 1.引言
### 1.1 编写目的
本测试报告为WIFI探针的商业大数据分析平台的测试报告，目的在于测试WIFI探针的商业大数据分析平台的稳定性，发现设计结构以及程序中的问题，本测试报告主要运用了一些常用的测试方法，黑盒测试和白盒测试，尽可能的发现WIFI探针的商业大数据分析平台的一些缺陷，不断的改善WIFI探针的商业大数据分析平台。
此报告预期读者：软件测试人员。
### 1.2 背景
说明： 

 - 所从属的软件系统的名称：WIFI探针的商业大数据分析平台
 - 项目提出者：中国软件杯组委会
 - 本项目的任务开发者：山东科技大学 计算机学院
 - 完成测试计划之前必须完成项目的需求分析、概要设计等工作。
 - 用户：商业软件使用人员 


###1.3 定义 

测试用例：是为测试而设计的数据
黑盒测试：不考虑内部逻辑结构，主要针对软件界面和软件功能进行测试，考虑程序是否能适当地接收输入数据而产生正确的输出信息
白盒测试：主要检查程序的内部结构，从检查程序的逻辑着手，得出测试数据。
